# Importation-of-products

Importing products based on regional tax rates, using enumerations, polymorphism for batch printing of tax information, interfaces with default implementations when necessary, and enumerations for code organization.
Importação de produtos baseado na taxação de imposto de cada região, utilizando enumerações, polimorfismo para imprimir taxação em lotes, interfaces com implementações default quando necessário, e enumerações para organização do código
